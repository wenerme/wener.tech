Wener's APIs

January 28, 2020

#Wener's APIs

  • apis.wener.me
    • Online version
    • NextJS SSR
    • Deployed on Vercel
  • wenerme.github.io/apis/lite.html
    • Lite version
    • Pure JS
    • Module Management

#概述

  • @wener/apis-server
    • 通过 now 部署的服务
    • 主要用于验证一些实验性质的想法和实现一些常用接口
    • 前端 react、antd、redux
    • 后端 nextjs、typeorm、pg
    • 特性
      • WebTorrent Web 客户端
      • Kong 网关管理界面
      • 摘要哈希计算
      • 二维码生成和解析
      • 条形码生成和解析
      • 电话归属地查询
  • @wener/utils / 文档
    • 常用的工具函数
    • 外部一些实用工具迁移为 ts，统一打包减少依赖
  • @wener/tinyrpc / 文档
    • 基于 ES6 代理的简洁的 RPC 实现
  • rjsf-antd-theme
    • 基于 jsonschema 生成 antd 主题 form 表单

#开发

  • 使用 Node 12 LTS 版本
# 如果使用 nvm
nvm install 12
nvm use 12

#Bundle

# ncc 构建单文件
ncc build scripts/setup-db.ts
# 执行
node dist/index.js
# pkg 打包 next 应用
yarn build
pkg -t node12-macos-x64 . --out-dir dist
# 执行服务端生成脚本
TS_NODE_PROJECT=$PWD/tsconfig.ts-node.json node -r ts-node/register -r tsconfig-paths/register server/prebuild.ts

#SCEL

# update index file
sqlite3 sougou-dict-cache.sqlite -Ax scel/index.full.json
ipfs files write -etp /scel/index.full.json scel/index.full.json
# get new hash
ipfs files stat /scel

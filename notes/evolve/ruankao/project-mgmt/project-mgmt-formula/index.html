三点估算#

info
  • Three-Point Estimating - 三点估算
  • PERT - Program Evaluation and Review Technique - 计划评审技术
  • 用于评估 成本 和 时间
    • 帮助规划进度
    • 帮助优化资源
  • 临近 beta 分部公式
  • 1:4:1
PERT=(O+4M+P)6=(to+4tm+tp)6PERT = \frac{(O + 4M + P)} {6} = \frac{(t_o + 4 t_m + t_p) } {6}PERT=6(O+4M+P)​=6(to​+4tm​+tp​)​
SD=tp−to6SD = \frac{t_p - t_o} {6}SD=6tp​−to​​
  • P - Pessimistic - 悲观值、最差的值
  • O - Optimistic - 乐观值、最好的值
  • M - Most likely - 通常值
  • PERT vs CPM
    • PERT
      • 面向事件
      • 估算的时间不确定 - 概率
      • 结束开始关系
      • 用于时间不具体时 - 单个节点变化不影响整体
    • CPM
      • 面向活动
      • 估算的时间是确定的
      • 可以有多种依赖关系
      • 任意节点时间变化影响最终估算
  • 参考
    • https://pmstudycircle.com/pert-program-evaluation-and-review-technique/
    • https://projectmanagementacademy.net/resources/blog/a-three-point-estimating-technique-pert/

净现值#

info
  • NPV - Net Present Value - 净现值
    • 也叫 NPW - Net Present Worth
    • 现值的总和
    • 金融学 相关词汇
    • 体现 金钱的时间价值(time value of money, TVM)
    • 用于项目评估收益
  • PV/Present Value/现值
PV=Rt(1+i)tPV=\frac{R_t}{(1+i)^t}PV=(1+i)tRt​​
NPV(i,N)=∑t=1NRt(1+i)tNPV(i,N)=\sum_{t=1}^{N}\frac{R_t}{(1+i)^t}NPV(i,N)=t=1∑N​(1+i)tRt​​
  • t - 时间 - 从 0 开始 - 例如 年、期
  • i - discount rate - 贴现率、折现率
  • RtR_tRt​ - net cash flow - 净现金流量
  • N - 持续时间
  • Net present value

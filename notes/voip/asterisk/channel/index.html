# 所有注册的通道类型
core show channeltypes

通道类型#

  • chan_mobile 和 chan_dongle 没显示 - 因为注册失败
  • chan_oss、chan_alsa、chan_console 只能注册一个 - 因为在做同一件事
  • SIP 功能可由 chan_sip 和 chan_pjsip 提供 - 目前建议使用 pjsip
  • IAX 是 asterisk 之间的通讯协议 - 用于实现多实例，互相发现，外部线路等

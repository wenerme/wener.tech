#U-Boot

#Ttips

  • Das U-Boot -- the Universal Boot Loader
    • Manual
    • wikipedia Das U-Boot
  • 主要用于嵌入式系统的开机加载程序,可以支持多种不同的计算机系统结构.
  • 支持架构
    • 68k, ARM, Blackfin, MicroBlaze, MIPS, Nios, SuperH, PPC, RISC-V, x86

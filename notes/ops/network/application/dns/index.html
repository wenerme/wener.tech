  • TSIG - Transaction Signature
  • KSK - Key Signing Key
    • 是一对公钥私钥.
    • 私钥用于生成 ZSK 的数字签名. 公钥存储于 DNS, 用于对 ZSK 进行认证
  • ZSK - Zone Signing Key
    • 是一对公钥私钥.
    • 私钥用于生成数字签名, 即为每个 RRSET 生成 RSSIG(Resource Record Signature).公钥存储于 DNS 用于对 RRSIG 进行认证.
    • 每个 DNSSET 中签名的 zone 都由 RRSIG 保护

# 物理设备
iw dev

iw dev wlan0 link
# AP 信息统计
iw dev wlan0 station dump

iw wlan0 connect foo
# 查看设备
iw list
# 热点扫描
iw dev wlan0 scan

# -t tuming
# -f frames
iw event

iw phy phy0 interface add moni0 type monitor

#adhoc vs ap

  • adhoc
    • 没有接入点 - P2P - 点对点连接
    • 每一个节点都相当于路由
    • 类似于蓝牙
    • 用于临时互联
    • Wi-Fi Direct 基于 adhoc
  • ap - Infrastructure
    • 桥接网络
    • 工作性质类似于交换机
    • 数据通过一个节点进行分发传输
  • Wireless access point vs. ad hoc network

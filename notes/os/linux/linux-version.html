Linux 4.7â€‹

Linux 4.6â€‹

  • USB 3.1 SuperSpeedPlus (10 Gbps) support
  • Improve the reliability of the Out Of Memory task killer
  • Support for Intel memory protection keys

  • Support for cgroup namespaces

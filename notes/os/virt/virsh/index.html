Virsh

Tips#

# 开机virsh start {domain}# 关机、断电virsh destroy {domain}# 触发关机 - 需要支持 ACPIvirsh shutdown {doamin}# 重启 - 需要支持 ACPIvirsh reboot {doamin}
# 运行的主机列表virsh list# 所有的主机列表virsh list --all
# 控制台访问# CTRL+] 退出virsh console DOMAIN
# 定义主机virsh define domain.xml
# 重命名# 如果通过 xml 需要 undefine 再 definevirsh domrename {domain} {new-name}
# 内存状态virsh dommemstat {domain}

clone#

virt-clone --original base --name k3s --file /data/vm/images/k3s.qcow2

migration#

  • https://libvirt.org/migration.html
virsh migrate web1 qemu+ssh://desthost/system

FAQ

error from service: CheckAuthorization: The name org.freedesktop.PolicyKit1 was not provided by any .service files#

  • qemu+ssh://admin@host/system - 可能是没有权限

/etc/libvirt/libvirtd.conf

# 允许分组访问unix_sock_group = "libvirt"unix_sock_rw_perms = "0770"
# 添加用户到分组usermod -G libvirtd -a username
# 验证virsh -c qemu:///system list

Unable to get DBus system bus connection: Failed to connect to socket /var/run/dbus/system_bus_socket: No such file or directory#

  • 确保 dbus 启动
  • 可能还需要重启 libvirtd

error from service: CheckAuthorization: The name org.freedesktop.PolicyKit1 was not provided by any .service files#

apk add polkit

sudo virsh hang#

killall -9 dmidecode

virsh list hang#

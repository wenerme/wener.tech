FAQ

GCC vs LLVM/Clang#

  • GCC
    • 使用更多
    • 支持 Fortran
    • 包可能使用 Clang 构建
  • LLVM/Clang
    • 只使用 LLVM 工具 - LLD linker, LIBC++
    • Clang 支持 ASAN
    • 原生支持 TLS/Thread-local storage
    • LLD 比 LD 更快 但功能特性更少
    • 缺少部分 GNU 下工具
    • Windows 10 支持 ARM64/AArch64 架构

MSVCRT vs UCRT#

  • MSVCRT - Microsoft Visual C++ Runtime
    • Windows 默认，后向兼容，非 C99 兼容
    • 不支持 UTF-8 locale
  • UCRT - Universal C Runtime
    • 新版本 Microsoft Visual Studio 默认
    • Windows 10 默认带

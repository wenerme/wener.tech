BrowserView vs webview#

  • BrowserWindow - 用于替代 webview 标签
    • Electron 接口 - 非 DOM - 位置相对于父窗口，需要自行管理
    • 不开启额外渲染进程 - 主进程控制
  • webview - 不推荐使用
    • html tag - 好使用，好布局
    • 额外渲染进程 - out-of-process iframe - 所有通讯都是 IPC
    • 速度慢于 iframe
  • 参考
    • web-embeds

WAL-G

  • 是什么？
    • PostgreSQL 备份、恢复工具
    • WAL-E 后继 - Golang 重写 Python - 性能 x4
    • 支持 PostgreSQL, MySQL/MariaDB, Mongo
    • 支持 S3, GCS, Azure, Swift, 本地, SSH 存储
  • wal-g/wal-g
    • Apache 2.0, lzo 部分 GPL 3.0+
    • PostgreSQL
  • WAL-E 不支持关闭 S3 SSE - SelfHost 常见不便于使用

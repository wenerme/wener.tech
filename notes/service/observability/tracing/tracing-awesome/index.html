FAQ

Jaeger vs Zipkin#

  • Jaeger
    • Golang, CNCF 项目
    • 更适用于 Kubernates 环境
    • 分为 agent、collector、query 等角色
    • 默认 0.1% 采样
    • spring-cloud-sleuth#628 - Jaeger integration
      • 可通过 brave-opentracing 来使用，但会丢失一些 jaeger 的特性
  • Zipkin
    • Java
    • 在容器化潮流前便已存在
    • 单服务，报表、采集
    • Spring Cloud Sleuth 有集成

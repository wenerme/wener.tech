core stack size#

  • PTHREAD_STACK_MIN
    • musl 下为 2048
  • FLB_CORO_STACK_SIZE=((3 * PTHREAD_STACK_MIN) / 2)
    • musl 下为 3072 - 是无效的 core stack size
    • 可以考虑设置为 24576
// 默认#define FLB_CORO_STACK_SIZE      ((3 * PTHREAD_STACK_MIN) / 2)

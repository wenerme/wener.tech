archivebox

  • ArchiveBox/ArchiveBox
    • MIT, Python
    • 针对唯一 URL 唯一提取类型 进行 Archive
    • 使用 chromium/chrome 生成 截屏、PDF、DOM HTML、headless JS
    • 使用 node & npm 生成 Readability, Mercury, SingleFile
    • 使用 wget 生成 纯 HTML, 静态文件, WARC
    • 使用 curl 获取 头、favicon、请求 archibe.org
    • 使用 youtube-dl 处理视频、音频、字幕
    • 使用 git 克隆仓库
  • 输出格式
    • Chrome SingleFile
      • https://github.com/gildas-lormeau/SingleFile
    • Chrome PDF
    • Chrome Screenshot
    • Wget HTML
    • Archive.org
    • Original
    • Headers
    • Chrome HTML
    • Readability
      • https://github.com/mozilla/readability
    • Mercury
    • Media
    • Git
  • ArchiveBox/ArchiveBox/docker-compose.yml
caution
  • 直接预览会执行 JS - 可能导致信息泄漏
    • 只有 wget 会提取 js，可以关闭集成
  • 避免使用 EXT3/FAT 之类的 FS - 对单目录有 50k 的限制
  • 不支持抓全站 - 目前 depth=0/1 ArchiveBox/ArchiveBox#191
  • 不支持单 URL 多快照 #179
    • 目前通过添加 Hash 实现重写快照
  • 没有 rest 接口 #496
docker run --rm -it \  -v $PWD/archivebox:/data -p 8000:8000 \  --name archivebox archivebox/archivebox# 创建管理员账号docker exec -it -u archivebox archivebox archivebox manage createsuperuser

IPFS

Tips#

  • the InterPlanetary File System
  • ipfs
  • ipfs.pics
    • Distributed image hosting
  • ipfstube
  • Why
    • 提供块级去重, 使得总体数据量得到压缩
    • 参考
      • Comparison of IPFS and BitTorrent for Archives
      • How does IPFS compare with X?
  • 端口
    • 4001 - Swarm TCP
    • 4002 - Swarm uTP
    • 5001 - API
      • /webui Web 管理端
    • 8080 - Gateway
    • 8081 - Swarm Websockets
  • 参考
    • Is it possible to use IPFS for LIVE video streaming?
  • IPFS 公共网关检测
  • 优势
    • 支持 gateway - 可以通过 HTTP 直接访问
    • 支持 MFS/虚拟目录
    • 目标是替代 internet 而不是单纯的 P2P 文件共享
    • ipns 替代 dns，ipfs 替代 internet
  • 劣势
    • IPFS 网络相对 torrent 没那么成熟
      • 意味着节点少
      • 资源少
    • 文件分发较慢
    • 开发公司重心转向 filecoin
  • 参考
    • https://github.com/cloudflare/ipfs-ext

# 启动服务端ipfs daemon
# 可使用 --api 指定远程接口# 查看节点ipfs --api /ip4/127.0.0.1/tcp/5001 swarm peers

# 启动服务# -e IPFS_LOGGING=debug 显示调试日志docker run -it --rm \   -v $PWD/repo:/data/ipfs \   -p 8080:8080 -p 4001:4001 -p 5001:5001 \   --name ipfs_host ipfs/go-ipfs

gqlgen

  • 99designs/gqlgen 是什么？
    • 基于生成的 GraphQL 引擎
    • Schema first - 需要 DSL 定义 GraphQL Schema
    • 类型安全
    • 支持 Plugin - 修改配置, 生成代码
  • 缺点
    • 库代码质量一般
    • 文档少
    • 插件能力弱，且没什么 Demo
  • gqlgen vs gophers vs graphql-go vs thunder
    • gqlgen 最近开发不活跃 - 2021-08
    • graphql-go/graphql 开发不活跃
caution
  • ⚠️ fragment 内的 alias 不会处理 - 非常严重且难排查
    • 99designs/gqlgen#1271
  • embeded 相同 struct 需要重复写 resolver
    • 99designs/gqlgen#592 - resolve 逻辑不能定义在 interface 上
    • 99designs/gqlgen#626 - 生成类型不支持 embedded
  • enum 不支持 int - 99designs/gqlgen#366
  • 99designs/gqlgen#1516
    • 不兼容 gqlparser 2.2 - 升级的时候注意
  • 99designs/gqlgen#1031 federation 只支持单个 @key
go get -u github.com/99designs/gqlgen
go run github.com/99designs/gqlgen init

go run github.com/99designs/gqlgen generate

srv.AroundOperations(func(ctx context.Context, next graphql.OperationHandler) graphql.ResponseHandler {
  if !isUserAdmin(ctx) {
    graphql.GetOperationContext(ctx).DisableIntrospection = true
  }
  return next(ctx)
})
